# gondola-finance-subgraph
The Graph exposes a GraphQL endpoint to query the events and entities within Gondola Finance.

# Schema
- Deposit
- EmergencyWithdraw
- OwnershipTransferred
- Withdraw

# Setup

    npm install -g @graphprotocol/graph-cli
    npm run codegen

# Adding an entity

- subgraph.yaml
- schema.graphql
- src/mapping.ts
